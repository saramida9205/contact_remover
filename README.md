# 엑셀 연락처 자동 삭제 도구 (Contact Remover)

<p align="center">
다수의 엑셀 파일(.xlsx, .xls) 내에 포함되어 있는 <strong>'연락처, 전화번호, 휴대폰, 핸드폰, 전화'</strong> 등과 관련된 민감 정보 컬럼을 식별하고 <strong>자동으로 일괄 삭제</strong>해주는 윈도우용 자동화 응용 프로그램입니다.
</p>

## ✨ 핵심 기능 (Features)
1. **고속 대용량 & 다중 시트 파싱**: 최신 Rust 기반 `Calamine` 엔진과 `Openpyxl` 혼합 적용을 통해, 수많은 시트를 가진 대용량 엑셀 파일이라도 압도적으로 빠른 로딩 속도를 보장합니다.
2. **유연한 예외 헤더 인식**: 데이터 테이블의 구조가 비정형이거나 헤더가 첫 줄(Row 1)이 아니더라도 상단 15줄 정도를 유연하게 스캔해 해당 연관 키워드를 식별 후 정확하게 삭제합니다.
3. **윈도우 우클릭 통합 병합 (IPC Socket 통신)**: 여러 파일을 드래그하여 바로 우클릭(연락처 자동 삭제 실행) 하셔도, 눈에 보이지 않는 찰나의 순간(0.5초) 동안 가장 처음 켜진 단일 파이썬 프로세스(마스터 창)가 나머지 프로세스들의 파일 경로 인자를 네트워크 소켓(로컬 통신)을 기반으로 모두 넘겨받아 취합하여 처리합니다. ('보내기'와 동일한 일체형 프로그래스 동작)
4. **성공 및 오류 상세 텍스트 기록 (자동 팝업)**: 오류가 걸려 패스된 엑셀암호화 파일 내역과, 정상 조치된 성공 결과 리스트를 별도의 텍스트(`작업성공_로그.txt`, `작업실패_로그.txt`) 파일에 나누어 생성하고 즉시 화면에 강제로 띄워 작업 결과를 직관적으로 파악할 수 있게 해줍니다.
5. **시간 정보 복원 기술**: 생성되거나 수정된 파일은 윈도우 파일 시스템 내부의 '메타데이터(만든 날짜/수정 날짜/액세스한 날짜)'가 훼손되지 않도록, 원본의 시계열 정보를 신규 생성된 파일에 그대로 위변조 복원 보장합니다.

## 📦 파일 구성 (Contents)
- **`contact_remover.py`** : (Python 소스 파일) Tkinter GUI 구현, IPC 프로그래스바 루프 및 엑셀 데이터 분석 코어 모음
- **`installer.iss`** : 배포용 **Inno Setup Script** 파일. 윈도우 레지스트리에 소프트웨어를 삽입하여 '우클릭 / 백그라운드 폴더' 단축 메뉴를 자동 생성시킵니다.
- **`Output/ContactRemoverSetup.exe`** : 파이썬 환경 구축이 전혀 필요없는 이 프로젝트의 **최종 빌드 인스톨 설치 파일**입니다. (다운로드 후 1초 안에 바로 윈도우에 통합 설치 가능)

## 🚀 사용법 (How to Install & Run)
1. 상단의 리포지토리 파일 목록 중 `Output/ContactRemoverSetup.exe` 다운로드 후 더블 클릭하여 설치 진행 (1초 소요).
2. 설치가 완료되면, 삭제하려는 엑셀 파일(혹은 폴더)들을 다중 드래그 선택하세요.
3. 원하는 만큼 선택한 뒤에 마우스 오른쪽 우클릭 버튼을 누르고 **[ 연락처 자동 삭제 실행 ]** 메뉴 클릭.
4. 모든 다중 파일들이 단 하나의 통합 응용프로그램 화면 속 병합 대기명단에 삽입됩니다. 
5. 해당 프로그램 화면 하단의 **[ 작업시작 ]** 버튼을 클릭하시면 프로그래스바와 함께 모든 민감 데이터 제거 작업이 즉시 개시됩니다.

---
> 해당 저장소의 소스코드는 **`PyInstaller`** 스탠드얼론 형태로 빌딩 하여 묶였고 **`Inno Setup 6`** 를 통해 Windows 10 & 11 최적화 통합 배포 본으로 자동 출력 처리되었습니다.
