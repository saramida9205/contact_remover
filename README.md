# 엑셀 연락처 자동 삭제 및 마스킹 도구 V2 Pro (Contact Remover Pro V2)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/Version-2.0_Pro-blue.svg)]()

이 도구는 엑셀 파일 내의 민감한 연락처 정보를 자동으로 식별하여 **열 전체를 삭제**하거나, 원본 데이터를 유지하면서 **중간 번호를 마스킹(****)** 처리하는 윈도우용 유틸리티입니다.

## 주요 기능 (V2 Pro 업데이트)

- **[NEW] 연락처 마스킹(****) 기능**: 연락처 열을 삭제하지 않고, 데이터 내부의 전화번호 패턴을 찾아 `010-****-5678` 형태로 안전하게 변환합니다.
- **[NEW] 데이터 기반 자동 감지**: 열 이름에 '연락처' 등의 키워드가 없어도, 셀 내부의 전화번호 패턴을 **최대 100행까지 자동 스캔**하여 마스킹 대상 열을 식별합니다. (헤더가 불분명한 파일 완벽 대응)
- **[NEW] 0 생략 번호 보정**: 엑셀에서 앞의 0이 빠진 번호도 정확히 인식하고, 마스킹 시 `010-****-6210` 형식으로 자동 보정합니다.
- **[NEW] 실행 속도 최적화**: 빌드 방식을 개선하여 프로그램 실행 시 발생하던 **로딩 지연 시간을 획기적으로 단축**했습니다. (아이콘 클릭 즉시 실행)
- **기존 연락처 열 삭제**: 헤더(연락처, 전화번호 등)를 지능적으로 분석하여 해당 열 전체를 제거합니다.
- **지능형 헤더 인식**: 데이터가 첫 줄에 없어도 상단 50줄을 스캔하여 정확한 헤더 위치를 찾아냅니다.
- **멀티스레딩 지원**: 대용량 작업 중에도 프로그램이 멈추지 않으며 실시간 진행률을 표시합니다.
- **윈도우 통합**: 파일 우클릭 메뉴 및 '보내기(SendTo)' 메뉴에 등록하여 즉시 실행 가능합니다.

## 설치 방법

1.  [최신 설치 파일 다운로드](Output/ContactRemoverSetup_V2Pro.exe) (리포지토리 내 Output 폴더)
2.  `ContactRemoverSetup_V2Pro.exe` 실행 및 설치
3.  설치 완료 후 엑셀 파일을 우클릭하면 '연락처 자동 삭제 및 마스킹 도구 실행' 메뉴가 나타납니다.

## 사용 방법

- **파일 우클릭**: 엑셀 파일(들)을 선택 후 우클릭 메뉴에서 실행합니다.
- **보내기 메뉴**: `우클릭 -> 보내기 -> 연락처 자동 삭제 및 마스킹 도구 실행`을 통해 실행합니다.
- **작업 모드 선택**:
    - **작업시작(연락처 열 삭제)**: 연락처 데이터가 포함된 열 전체를 삭제합니다. (열이 사라짐)
    - **작업시작(연락처 **** 변경)**: 연락처 데이터의 중간 번호를 별표로 치환합니다. (열은 유지됨)
