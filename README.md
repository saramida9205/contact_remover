# 엑셀 연락처 자동 삭제 도구 (Contact Remover Pro)

<p align="center">
다수의 엑셀 파일(.xlsx, .xls) 내에 포함되어 있는 <strong>'연락처, 전화번호, 휴대폰, 핸드폰, 전화'</strong> 등과 관련된 민감 정보 컬럼을 식별하고 <strong>자동으로 일괄 삭제</strong>해주는 윈도우용 고성능 자동화 응용 프로그램입니다.
</p>

## ✨ 주요 최적화 기능 (Pro Version)
1. **백그라운드 멀티스레딩 (Responsive UI)**: 파일 처리 로직을 메인 GUI와 완전히 분리하여, 수만 건의 대용량 데이터를 처리하는 중에도 프로그램이 멈추지 않고 창 이동 및 상태 확인이 실시간으로 가능합니다.
2. **작업 중단(Cancel) 기능**: 작업 도중 실수나 변동 사항이 생길 경우, 즉시 안전하게 현재 작업을 멈출 수 있는 전용 중단 버튼을 제공합니다.
3. **원자적 저장(Atomic Save) & 데이터 보호**: 임시 파일(`.tmp`) 기반의 저장 방식을 채택하여, 작업 중 갑작스러운 전원 차단이나 오류가 발생하더라도 원본 파일이 절대 훼손되지 않도록 설계되었습니다.
4. **지능형 헤더 인식 기술**: 상단 15행을 지능적으로 스캔하여 데이터 본문의 시작 위치를 정확히 판별하므로, 비정형 엑셀 양식에서도 높은 연락처 열 인식률을 보장합니다.
5. **고속 엑셀 파싱**: 최신 Rust 기반 `Calamine` 엔진과 `Openpyxl`을 혼합 사용하여 압도적인 읽기/쓰기 속도를 제공합니다.
6. **윈도우 통합 UX**: 우클릭 메뉴 및 '보내기' 메뉴 연동을 통해 여러 파일을 한 번에 선택하여 단일 UI에서 일괄 처리할 수 있는 IPC 소켓 통신 병합(Single Instance) 기능을 지원합니다.
7. **시간 정보 및 메타데이터 복원**: 파일 수정 후에도 원본의 '만든 날짜/수정 날짜/액세스 날짜'를 동일하게 위변조 복원하여 데이터 신뢰성을 유지합니다.

## 📦 파일 구성 (Contents)
- **`contact_remover.py`** : (Python 소스 파일) UI-로직 분리형 Clean Code로 구현된 Pro 버전 소스
- **`installer.iss`** : 윈도우 우클릭 메뉴 자동 등록 기능을 포함한 **Inno Setup Script**
- **`Output/ContactRemoverSetup.exe`** : **최종 빌드된 Pro 버전 설치 파일** (즉시 사용 가능)

## 🚀 사용법 (How to Install & Run)
1. `Output/ContactRemoverSetup.exe` 다운로드 후 실행하여 1초 만에 설치 완료.
2. 삭제하려는 엑셀 파일들을 드래그하여 마우스 오른쪽 클릭 -> **[ 연락처 자동 삭제 실행 ]** 클릭.
3. 프로그래스바가 있는 고성능 통합 화면이 나타납니다.
4. **[ 작업시작 ]** 버튼을 클릭하여 안전하고 빠른 민감 정보 제거를 개시하세요.
5. 필요 시 **[ 작업중단 ]** 버튼을 사용하여 즉시 멈출 수 있습니다.

---
> 해당 저장소는 **`PyInstaller`** 및 **`Inno Setup 6`** 로 패키징 되었으며, Windows 10/11 환경에 최적화된 최신 Pro 빌드 배포판입니다.
